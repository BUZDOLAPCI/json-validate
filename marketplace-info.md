# Dedalus Marketplace Info

## Short Description

## Long Description
Validate any JSON against JSON Schema with detailed error reporting, get human-readable explanations of validation failures, and automatically repair malformed or invalid JSON. Tools: validate_json, explain_validation, repair_json.
